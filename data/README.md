# Dataset Information

This project uses the Online Retail dataset.

- The full dataset contains over 500,000 rows.
- Due to size limitations, only a sample of 100 rows is included here.

## Files

- `sample_data.csv` → Small subset of the dataset for reference and reproducibility

## Notes

- Data includes transactions, product details, and customer IDs
- Negative quantities represent returns
- Invoice numbers starting with "C" indicate cancellations
